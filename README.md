# 🎯 Student Performance Analysis

## 🔍 Overview
This project analyzes student data to understand the key factors affecting academic performance.  
The analysis includes data cleaning, visualization, and performance categorization.

---

## 📊 Objectives
- Analyze student performance trends  
- Identify factors affecting scores  
- Categorize students based on performance  
- Generate insights using data visualization  

---

## 📁 Dataset
The dataset contains information about students including:
- Gender  
- Age  
- Department  
- Attendance  
- Midterm & Final Scores  
- Participation & Project Scores  

---

## ⚙️ Process
1. Data Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Visualization using charts  
4. Performance classification  

---

## 📈 Key Insights
- Students with higher attendance perform better  
- Participation score significantly impacts final results  
- Academic performance varies across departments  
- Consistent students achieve higher scores  

---
